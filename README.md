# Auto-Correct-Keyboard-NLP
Auto Correct Keyboard using NLP | Flask + T5 Model for real-time grammar correction with confidence score
Auto Correct Keyboard (NLP Project)

This project is a web-based Auto Correct Keyboard that uses a T5 Transformer model to correct grammar and improve sentence structure.

Features:
Real-time grammar correction
NLP-based prediction
Confidence score display
Clean UI

Tech Stack:
Python (Flask)
Transformers (Hugging Face)
HTML, CSS, JavaScript
